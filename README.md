## Transformaciones aplicadas

1. **Resumen sobrevivientes (Titanic):** tabla con conteo de pasajeros que sobrevivieron o no.
2. **UniqueKey en Libros:** columna con UUID único por cada libro.
3. **Promedio de temperatura (Clima):** promedio calculado y almacenado en almacen_datos.
4. **Filtrado Titanic:** se eliminaron pasajeros menores de 10 años.
